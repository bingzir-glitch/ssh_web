# 网页 SSH 控制台

一个基于浏览器的 SSH 运维项目，用来把远程终端、文件浏览和常用连接管理集中到一个页面里。

## 功能

- 网页内直接连接 SSH 服务器
- 支持实时终端操作
- 支持远程文件浏览与基础管理
- 支持保存常用连接，默认从后端文件读取
- 支持夜间模式和日间模式切换

## 技术栈

- 前端：Vue 3、Vite、xterm.js
- 后端：ASP.NET Core、WebSocket、SSH.NET
- 存储：JSON 文件

## 目录

- `ssh.client`：前端界面
- `ssh.Server`：后端服务

## 简介

这是一个偏轻量的网页 SSH 工具，适合日常服务器连接、命令执行和文件管理场景。
